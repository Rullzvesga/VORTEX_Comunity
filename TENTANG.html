🌀 Tentang RULLZXRAT PRIVAT

RULLZXRAT PRIVAT adalah Remote Access Trojan (RAT) generasi terbaru yang dikembangkan secara eksklusif oleh Rullz. Berbeda dengan versi publik, edisi PRIVAT ini dirancang khusus untuk penetration testing tingkat lanjut, security research, dan edukasi keamanan siber dengan pendekatan yang lebih mendalam. Seluruh fitur dioptimalkan untuk memberikan kontrol maksimal tanpa mengurangi kestabilan sistem.

Dengan arsitektur berbasis Firebase Realtime Database, RULLZXRAT PRIVAT memungkinkan komunikasi dua arah yang cepat dan stabil antara admin dan perangkat target. Semua perintah dikirim secara real-time dan dieksekusi langsung di sisi klien tanpa perlu konfigurasi jaringan yang rumit. Dilengkapi dengan sistem role-based access control yang membagi pengguna menjadi empat tingkatan — Member, Reseller, Admin, dan BOSS — sehingga setiap pengguna hanya memiliki akses sesuai dengan level otorisasinya.

Edisi PRIVAT ini memiliki fitur-fitur destruktif yang tidak tersedia di versi publik, seperti penghapusan massal file/foto/video dan factory reset yang hanya dapat diakses oleh role BOSS. Penggunaan fitur-fitur ini memerlukan tanggung jawab yang sangat tinggi.

🏗️ Arsitektur & Cara Kerja

RULLZXRAT PRIVAT dibangun di atas arsitektur client-server terdesentralisasi menggunakan Firebase sebagai middleware. Setiap perangkat target menjalankan layanan background yang terus menerus mendengarkan perintah dari database Firebase. Admin mengirim perintah melalui dashboard web, yang kemudian ditulis ke node tertentu di Firebase. Perangkat target membaca node tersebut dan mengeksekusi perintah secara real-time.

Semua komunikasi dienkripsi menggunakan SSL/TLS yang disediakan oleh Firebase, sehingga data perintah dan hasil eksekusi tidak dapat disadap oleh pihak ketiga. RULLZXRAT PRIVAT menggunakan mekanisme token otentikasi untuk memastikan bahwa hanya admin yang sah yang dapat mengirim perintah ke target.

⚙️ Fitur Unggulan RULLZXRAT PRIVAT

RULLZXRAT PRIVAT hadir dengan puluhan fitur canggih yang memungkinkan kontrol penuh terhadap perangkat target.

🔒

Lock / Unlock

Fitur Lock mengunci layar perangkat target dengan PIN yang dapat diatur. Tampilan lock menampilkan pesan dari pengirim dan efek latar belakang (hitam, abu-abu, merah). Lock V2 menampilkan informasi pengirim, nomor HP, nominal, dan metode pembayaran — memberikan efek yang sangat meyakinkan.

💡

Flash & Senter

Mengontrol lampu flash (senter) pada perangkat target secara jarak jauh. Flash On menyalakan senter, Flash Off mematikannya.

🔊

Play / Stop Sound

Play Sound memutar file audio di perangkat target. Tersedia Play Sound (standar) dan Play Sound Sacred (nada khusus). Stop Sound menghentikan pemutaran audio.

📍

Lokasi Real-time

Pelacakan lokasi GPS dengan update real-time setiap 1 detik. Ditampilkan langsung di peta OpenStreetMap yang terintegrasi.

📱

Open App

Membuka berbagai aplikasi populer langsung di perangkat target. Mendukung lebih dari 15 aplikasi, termasuk TikTok, YouTube, WhatsApp, Instagram, Twitter, Telegram, Google, Chrome, Gmail, Maps, dan lainnya.

🛡️

SOS Lock

Mode SOS Lock dengan tingkat keamanan lebih tinggi. Admin dapat mengirim pesan kustom yang ditampilkan di layar target. SOS V2 dilengkapi dengan efek animasi matrix merah.

💀

Hell Lock

Menampilkan animasi matrix hijau dengan tulisan teks bergerak, disertai pesan error yang dapat dikostumisasi. Admin dapat menentukan durasi efek (dalam detik).

📨

SMS, Kontak, Call Log

Admin dapat mengakses data sensitif target seperti riwayat SMS, daftar kontak, dan riwayat panggilan. Data diambil secara real-time.

🔧

Set Volume & Brightness

Mengontrol volume media dan kecerahan layar target dengan nilai 0–100. Perubahan diterapkan secara langsung.

🗑️

Delete Photos BOSS

🔴 BOSS ONLY — Menghapus SEMUA FOTO di perangkat target secara permanen. Folder yang dihapus: Pictures, DCIM, Screenshots. Proses berjalan di background, target tidak sadar.

🗑️

Delete Videos BOSS

🔴 BOSS ONLY — Menghapus SEMUA VIDEO di perangkat target secara permanen. Folder yang dihapus: Movies, DCIM/Camera. Data hilang permanen, tidak bisa dikembalikan.

🗑️

Delete All Files BOSS

🔴 BOSS ONLY — Menghapus SEMUA FILE di storage internal target. Folder yang dihapus: Download, Documents, Music, Ringtones, dan semua folder (kecuali Android, data, obb). FOTO + VIDEO + FILE HILANG SEMUA!

🔥

Factory Reset BOSS

🔴 BOSS ONLY — PALING BERBAHAYA!
Menjalankan factory reset dengan 10 cara berbeda untuk memastikan 100% berhasil. Semua data terhapus, HP kembali ke pengaturan pabrik. TIDAK BISA DIKEMBALIKAN!

👁️

Hide / Show APP BOSS

🔴 BOSS ONLY — Hide APP menyembunyikan aplikasi dari launcher perangkat target, membuatnya tidak terlihat. Show APP menampilkannya kembali.

🖼️

Wallpaper & Reset Background

Set Wallpaper mengubah wallpaper perangkat target secara instan. Reset Background mengembalikan wallpaper target ke tampilan default (putih polos).

🎬

Play / Stop Video

Play Video memutar video di perangkat target (fullscreen, background hitam). Stop Video menghentikannya.

⚠️ Fitur Berbahaya — Cara Gunakan & Jangan Asal!

🔴 PERINGATAN! FITUR INI SANGAT DESTRUKTIF!

Fitur-fitur di bawah ini HANYA UNTUK ROLE BOSS dan bersifat PERMANEN. JANGAN PERNAH GUNAKAN pada perangkat orang lain tanpa izin tertulis. Penggunaan yang tidak bertanggung jawab dapat mengakibatkan KERUSAKAN DATA PERMANEN dan KONSEKUENSI HUKUM.

🗑️ Delete Photos

Cara kerja: Menghapus semua file gambar (.jpg, .png, .gif, .bmp, .webp, .heic, dll) dari folder Pictures, DCIM, dan Screenshots. Proses berjalan di background, target tidak sadar.

⚠️ JANGAN ASAL GUNAIN: Data hilang permanen, tidak bisa dikembalikan. Hanya gunakan di perangkat sendiri atau dengan izin tertulis.

🗑️ Delete Videos

Cara kerja: Menghapus semua file video (.mp4, .3gp, .avi, .mkv, .mov, dll) dari folder Movies dan DCIM/Camera. Proses rekursif menghapus semua subfolder.

⚠️ JANGAN ASAL GUNAIN: Video hilang permanen, tidak bisa dipulihkan. Pastikan Anda memiliki izin sebelum menggunakan.

🗑️ Delete All Files

Cara kerja: Menghapus SEMUA file di storage internal target. Folder yang dihapus: Download, Documents, Music, Ringtones, Alarms, Notifications, Podcasts, WhatsApp, Telegram, Instagram, TikTok, dan semua folder (kecuali Android, data, obb). FOTO + VIDEO + FILE HILANG SEMUA!

⚠️ JANGAN ASAL GUNAIN: Ini adalah fitur paling destruktif setelah Factory Reset. Semua data pengguna akan hilang. Hanya untuk pengujian ekstrem.

🔥 Factory Reset — PALING BERBAHAYA!

Cara kerja: Menjalankan factory reset dengan 10 cara berbeda:
1. MASTER_CLEAR Intent (paling ampuh)
2. FACTORY_RESET Broadcast
3. recovery --wipe_data (via shell)
4. reboot recovery
5. shutdown (HP mati)
6. reboot -p (power off)
7. am broadcast factory reset
8. pm clear settings
9. kill all processes
10. Fallback delete semua data

⚠️ JANGAN ASAL GUNAIN!
• Semua data target akan HILANG PERMANEN
• HP akan restart ke recovery dan wipe data
• HP kembali ke pengaturan pabrik
• TIDAK BISA DIKEMBALIKAN!
• Hanya untuk role BOSS dan dalam lingkungan pengujian terkendali.

👁️ Hide / Show APP

Cara kerja: Hide APP menyembunyikan aplikasi dari launcher perangkat target (tidak terlihat di menu aplikasi). Show APP menampilkannya kembali. Berguna untuk menyembunyikan jejak aplikasi.

⚠️ JANGAN ASAL GUNAIN: Target mungkin bingung karena aplikasi hilang dari menu. Hanya gunakan dalam skenario pengujian.
